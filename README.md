# Dual-Potentiometer-DC-Fan-Controller

This repository contains code for my potentiometer DC Fan Controller. 
The project utilizes the PIC18F4321 with the MPLAB X IDE. I wired together a physical circuit with the PIC18F4321, LCD, and two potentiometers.
While programming this, I had to configure the registers so that analog inputs can be read. I also had to configure the registers for timers, interrupts, and GPIO.
